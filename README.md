## 📝 Sobre o Projeto

O **Saúde Humanizada** é uma aplicação web desenvolvida como **Projeto Extensionista** para a disciplina de **Desenvolvimento Web II**. O objetivo principal é facilitar o acesso a informações de saúde, unindo medicina convencional acessível, tratamentos naturais e orientações nutricionais em um só lugar.

---

## ✨ Funcionalidades Principais

* 🔐 **Autenticação de Usuários:** Páginas de Login e Cadastro integradas ao banco de dados.
* 🛡️ **Acesso Restrito:** Informações detalhadas sobre medicamentos são visíveis apenas para usuários logados, garantindo segurança na navegação.
* 💊 **Catálogo de Medicamentos:** Busca de remédios convencionais com foco em custo-benefício e informações técnicas.
* 🌿 **Medicina Natural:** Sugestões de alternativas naturais (chás, sucos e fitoterápicos) baseadas em sintomas como gripe, náuseas e infecções.
* 🍎 **Guia de Nutrição:** Planos alimentares específicos para condições como Diabetes e Gastrite, além da Dieta Mediterrânea.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído com foco em uma estrutura web completa (Fullstack):

* **Frontend:** HTML5, CSS3 e JavaScript.
* **Backend:** Node.js.
* **Banco de Dados:** Integração para armazenamento de usuários e consultas de medicamentos.
* **Estilo:** Design focado em usabilidade e "humanização" do atendimento visual.

---

## 📂 Estrutura do Projeto

* `public/` ou `views/`: Contém os arquivos de interface (HTML/CSS).
* `scripts/`: Lógica de frontend e validações em JavaScript.
* `server.js` (ou similar): Configuração do servidor Node.js e rotas.
* `database/`: Scripts de conexão e modelos do banco de dados.

## 🚀 Como Executar

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/stephanievitoria/Saude-Humanizada.git](https://github.com/stephanievitoria/Saude-Humanizada.git)
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Inicie o servidor:**
    ```bash
    npm start
    ```

4.  **Acesse no navegador:**
    `http://localhost:3000` (ou a porta configurada no seu projeto).
